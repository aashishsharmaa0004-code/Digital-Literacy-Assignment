digital-literacy-project

Digital Literacy Project is about learning important digital skills like making online profiles, using coding and collab platforms, doing professional communication and knowing about cyber safety. It helps students to become more confident and responsible while using digital technology.

1. Hands-on Training on Presentation Software
In this module we learn how to use tools like PowerPoint or Prezi for making good presentations. It includes slide designing, using visuals and explaining ideas in clear way.

2. Building a Digital Portfolio
This module focus on creating a online portfolio where we can show our skills, projects and achievements. It helps students to make a good professional profile for internships and jobs.

3. Programming Platforms & Collaborative Learning Platforms
Students were introduced with platforms like GitHub, Kaggle and HackerRank for coding practice, sharing projects and learning together.

4. Email Etiquette & Basics of Social Media Etiquette
This module explain about writing professional emails and using social media in a proper way. It includes tone, privacy and behaviour online.

5. Awareness on Cyber Crime and Financial Fraudulent
This part focus on cyber crimes like phishing, scams and frauds. Also teaches about safety like strong passwords and safe online payments.

✅ Overall: All these modules together helps in building digital skills, communication and awareness for safe and professional use of internet.
